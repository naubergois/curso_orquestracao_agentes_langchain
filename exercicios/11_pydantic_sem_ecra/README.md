# Exercício 11 — Pydantic (sem ecrã)

Notebook Jupyter sobre **Pydantic v2**: modelos, `Field`, validadores, modelos aninhados, serialização JSON e **JSON Schema**. Secção **opcional** liga o mesmo schema ao Gemini via `with_structured_output`.

## Arranque

```bash
./run.sh
```

Abrir `exercicio_11_sem_ecra.ipynb`. Variável opcional: `GEMINI_MODEL_EX11` (ver `.env.example` na raiz do repo, se existir entrada para este exercício).
