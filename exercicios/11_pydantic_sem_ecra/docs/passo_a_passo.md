# Passo a passo — Exercício 11

1. Copie `.env.example` para `.env` na **raiz do repositório** (opcional até à secção Gemini).
2. Entre em `exercicios/11_pydantic_sem_ecra` e execute `./run.sh` (ou `run.ps1` no Windows).
3. Abra `exercicio_11_sem_ecra.ipynb` e execute as células **por ordem**.
4. Parar: `docker compose -f docker-compose.jupyter.yml down` nesta pasta.
